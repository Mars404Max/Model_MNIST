# Number Guessing Model

A small Python project that contains a simple macchine learning model for guessing numbers. 
The repository includes multiple scripts: training, prediction and utility functions.

## Project Structure
- `draw_digit.py` - number drawing program
- `neural_net_test.py` - main script of the model
- `predict_digit.py` -  script that launches number guessing

## Usage

Run predictions:

```
python3 predict_digit.py
```
